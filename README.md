# Project Setup

## How to Open This Project

### Method 1: Open in Browser
1. Navigate to the project folder
2. Double-click on `index.html`
3. The file will open in your default web browser

### Method 2: Using VS Code Live Server
1. Open the project folder in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

### Method 3: Direct File Path
1. Open any web browser (Chrome, Firefox, Edge, etc.)
2. Press `Ctrl + O` (or `Cmd + O` on Mac)
3. Navigate to this folder and select `index.html`

## Project Structure
```
├── index.html      # Main HTML file
├── style.css       # Stylesheet
├── Collection/     # Assets folder
└── fonts/          # Font files
```

## Notes
- No installation required
- No server setup needed for basic functionality
- Simply open `index.html` in any modern web browser
